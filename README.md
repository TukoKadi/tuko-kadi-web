# Tuko Kadi Platform 🇰🇪

A community-driven, open-source tool to help every young Kenyan find their nearest registration center.

## Our Mission
- **No Politics:** This is 100% citizen-led.
- **No Tracking:** We do not collect ID numbers or phone numbers.
- **Crowdsourced:** Data is added by the community, for the community.

## How to Contribute
If you are a developer, feel free to fork this repo and submit a Pull Request. Let's build for Kenya.
